# Jasmine Apsari Portfolio 🚀

Personal portfolio website built with **React Native Web (Expo)** featuring a modern, minimalist "Black Pink" aesthetic.

## ✨ Features
- **Modern UI**: High-contrast Black & Pink theme with Poppins font.
- **Interactive Progress Bars**: Hover-responsive skill meters with custom glow effects.
- **Glassmorphism Navbar**: Floating pill-shaped navigation at the top.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop.
- **Custom Profile**: Features personal photo with interactive hover ring.

## 🛠️ Tech Stack
- **Framework**: Expo (React Native Web)
- **Styling**: StyleSheet (Vanilla CSS-in-JS)
- **Animations**: React Native Reanimated
- **Icons**: Expo Vector Icons (Ionicons)
- **Font**: Poppins (Google Fonts)

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Jasmine-Apsari/cv-ppt.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run web
   ```
