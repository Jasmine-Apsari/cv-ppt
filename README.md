## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Jasmine-Apsari/cv-ppt.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run web
   ```
